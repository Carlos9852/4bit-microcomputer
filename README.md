# MikroLeo
## 4-bit Didactic Microcomputer

The project is in the final testing stage.

Main Features:
- 2k x 16 ROM (up to 4k)
- 2k x 4 RAM (up to 4k)
- 16 OUTPUTS
- 16 INPUTS
- SINGLE CYCLE INSTRUCTION
- 3 EXECUTION MODES:
   * STEP
   * 3MHz (PRECISE TIME)
   * ADJUSTABLE CLOCK SPEED
- NO MPU/MCU OR COMPLEX
   CHIPS
- RISC ARCHITECTURE
- NO MICROCODE
- Single board with 295.9mm x 196.9mm
